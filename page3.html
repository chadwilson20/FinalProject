Q2: How many guesses did you have to use to get the right number? You should have only used 8 guesses at the max. Why do you think this is? (Hint: there is a mathematical reason for it).

A2:

Correct answer for Q2

How binary search works is that it is based off of a log base 2 system, since this is the binary logarithm. So let's say from our random number guessing game example that we have 200 numbers to search through. If you take the log base 2 of 200, you will see that it is approximatly equal to 8, which is the most number of guess that you will need to find the right number in 200 numbers using binary search.
